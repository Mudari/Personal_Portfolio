# Mudari Karthik Sai - Portfolio Website

A modern, responsive portfolio website showcasing the skills, projects, and achievements of Mudari Karthik Sai, a Full-Stack Developer.

## 🖼️ Adding Your Profile Photo

To add your profile photo to the portfolio:

1. **Replace the placeholder file**: Delete the current `profile-photo.jpg` file
2. **Add your photo**: Place your profile photo in the same directory with the name `profile-photo.jpg`
3. **Recommended specifications**:
   - **Size**: 300x300 pixels or larger (square aspect ratio)
   - **Format**: JPG, PNG, or WebP
   - **Quality**: High resolution for crisp display
   - **Style**: Professional headshot with good lighting

The photo will automatically appear in both the hero section and about section of your portfolio.

## Features

### 🎨 Design & UI
- **Clean & Minimalist Design**: Modern, professional appearance with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Dark/Light Mode**: Toggle between dark and light themes with persistent preference
- **Modern Typography**: Uses Poppins font family for excellent readability
- **Smooth Animations**: CSS animations and JavaScript-powered scroll effects

### 🚀 Interactive Features
- **Sticky Navigation**: Fixed navbar with smooth scrolling to sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Back to Top Button**: Appears when scrolling down
- **Download Resume**: Functional resume download button
- **Hover Effects**: Interactive hover animations on buttons and cards
- **Contact Links**: Direct links to email, phone, LinkedIn, and GitHub

### 📱 Sections
1. **Hero Section**: Introduction with name, title, and call-to-action buttons
2. **About Me**: Personal introduction with achievements and statistics
3. **Skills**: Progress bars showing technical skills proficiency
4. **Journey**: Timeline of career progression and achievements
5. **Projects**: Project cards with descriptions and links
6. **Contact**: Contact information with direct links

### 🛠 Technical Features
- **Semantic HTML5**: Well-structured, accessible markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **Vanilla JavaScript**: No frameworks, pure JavaScript for interactivity
- **Performance Optimized**: Fast loading with optimized assets
- **Cross-browser Compatible**: Works on all modern browsers

## Getting Started

1. **Clone or Download**: Get the project files
2. **Open in Browser**: Simply open `index.html` in any modern web browser

4. **Deploy**: Upload to any web hosting service
